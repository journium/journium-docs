## Setup

Set up Journium for your React application.

<section id="setup-section">
<div className="fd-steps [&_h3]:fd-step">

### Install `@journium/react`

The [Journium React SDK](/docs/react/hooks) gives you access to pre-built hooks and components to help you generate events 
in your React application and get insights from your data.

Run the following command to install the SDK:

```npm title="Terminal"
npm i @journium/react
```

### Set your Journium Publishable Key 

1. **Create a new application** in the [Journium dashboard](https://dashboard.journium.app). This is where you will get your API keys.

2. **Get your Publishable Key** from the **Developers | API Keys** section. 

3. Paste your key into your `.env` or `.env.local` file.

The final result will look like this:

<Tabs groupId="react-framework" items={['Vite projects', 'React/(CRA) projects']} persist>
  <Tab value="Vite projects">
  ```bash title=".env"
  VITE_JOURNIUM_PUBLISHABLE_KEY=your-publishable-key
  ```
  </Tab>
  <Tab value="React/(CRA) projects">

  ```bash title=".env"
  REACT_APP_JOURNIUM_PUBLISHABLE_KEY=your-publishable-key
  ```
  </Tab>
</Tabs>

### Add `<JourniumProvider>` to your app

The `<JourniumProvider>` component provides session and user context to Journium's hooks. 
It's recommended to wrap your entire app at the entry point with `<JourniumProvider>` 
to make analytics globally accessible throughout your application. 


<Tabs groupId="react-framework" items={['Vite projects', 'React/(CRA) projects']} persist defaultIndex={1}>
  <Tab value="Vite projects">
    ```tsx title="main.tsx"
    import { StrictMode } from 'react'
    import { createRoot } from 'react-dom/client'
    // [!code ++:1]
    import { JourniumProvider } from '@journium/react';
    import App from './App';

    createRoot(document.getElementById('root')!).render(
      <StrictMode>
        {/* [!code ++:5] */}
        <JourniumProvider
          config={{
            publishableKey: import.meta.env.VITE_JOURNIUM_PUBLISHABLE_KEY!
          }}
        >
          <App />
        {/* [!code ++:1] */}
        </JourniumProvider>
      </StrictMode>
    );
    ```
  </Tab>
  <Tab value="React/(CRA) projects">
    ```tsx title="main.tsx"
    import React from 'react';
    // [!code ++:1]
    import { JourniumProvider } from '@journium/react';
    import App from './App';

    function Root() {
      return (
        // [!code ++:5]
        <JourniumProvider
          config={{
            publishableKey: process.env.REACT_APP_JOURNIUM_PUBLISHABLE_KEY!
          }}
        >
          <App />
        {/* [!code ++:1] */}
        </JourniumProvider>
      );
    }

    export default Root;
    ```
  </Tab>
</Tabs>



### Done

You can now start your application and see the Journium SDK in action. If you navigate around in your application, 
the Journium SDK will push telemetry to the Journium API. You can see events in the Journium dashboard in the
**Developers | Events** section of your application. Head over to the 
[Journium dashboard](https://dashboard.journium.app) to see the events.
