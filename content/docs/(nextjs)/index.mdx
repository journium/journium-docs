## Setup

Set up Journium for your Next.js application.

<section id="setup-section">
<div className="fd-steps [&_h3]:fd-step">

### Installation

```npm
npm i @journium/nextjs
```


### Initialize Journium

Wrap your app wih the `NextJourniumProvider` to enable analytics throughout your Next.js application.

#### App Router

Add the `NextJourniumProvider` to your root layout, usuallay the `app/layout.tsx` file.

```tsx title="app/layout.tsx"
// [!code ++:1]
import { NextJourniumProvider } from '@journium/nextjs';

export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="en">
      <body>
         {/* [!code ++:1] */}
        <NextJourniumProvider>
          {children}
          {/* [!code ++:1] */}
        </NextJourniumProvider>
      </body>
    </html>
  );
}
```

#### Pages Router

Add the `NextJourniumProvider` to your pages router.

```tsx title="pages/_app.tsx"
import type { AppProps } from 'next/app';
// [!code ++:1]
import { NextJourniumProvider } from '@journium/nextjs';

export default function App({ Component, pageProps }: AppProps) {
  return (
    {/* [!code ++:1] */}
    <NextJourniumProvider>
      <Component {...pageProps} />
    {/* [!code ++:1] */}
    </NextJourniumProvider>
  );
}
```

### Get your Publishable Key

1. **Sign up for a Journium account** at [https://journium.app/signup](https://journium.app/signup)

2. **Create a new application** in the Journium dashboard. This is where you will get your API keys.

3. **Get your Publishable Key** from the **Developers | API Keys** section. 


### Set your Publishable Key 

**Set your Publishable Key** in your environment variables. 
The publishable key can be set in your `.env` or `.env.local` file.

```bash title=".env"
NEXT_PUBLIC_JOURNIUM_PUBLISHABLE_KEY=your-publishable-key
```

### Done

You can now start your application and see the Journium SDK in action. If you navigate around in your application, 
the Journium SDK will push telemetry to the Journium API. You can see events in the Journium dashboard in the
**Developers | Events** section of your application. Head over to the 
[Journium dashboard](https://dashboard.journium.app) to see the events.
