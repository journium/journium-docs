## Setup

Set up Journium for your Next.js application.

<section id="setup-section">
<div className="fd-steps [&_h3]:fd-step">

### Install `@journium/nextjs`

The [Journium Next.js SDK](/docs/hooks) gives you access to pre-built hooks and components to help you generate events 
in your Next.js application and get insights from your data.

Run the following command to install the SDK:

```npm title="Terminal"
npm i @journium/nextjs
```



### Set your Journium Publishable Key 

1. **Create a new application** in the [Journium dashboard](https://dashboard.journium.app). 
This is where you will get your API keys.

2. **Get your Publishable Key** from the **Developers | API Keys** section. 

3. Paste your key into your `.env` or `.env.local` file.

The final result will look like this:


```bash title=".env"
NEXT_PUBLIC_JOURNIUM_PUBLISHABLE_KEY=your-publishable-key
```

### Add `<NextJourniumProvider>` to your app

The `<NextJourniumProvider>` component provides session and user context to Journium's hooks. 
It's recommended to wrap your entire app at the entry point with `<NextJourniumProvider>` 
to make analytics globally accessible throughout your application. 


<Tabs groupId="nextjs-framework" items={['App Router', 'Pages Router']} persist defaultIndex={1}>
  <Tab value="App Router">
    
      ```tsx title="app/layout.tsx"
      // [!code ++:1]
      import { NextJourniumProvider } from '@journium/nextjs';

      export default function RootLayout({
        children,
      }: {
        children: React.ReactNode;
      }) {
        return (
          <html lang="en" suppressHydrationWarning>
            <body>
              {/* [!code ++:1] */}
              <NextJourniumProvider>
                {children}
                {/* [!code ++:1] */}
              </NextJourniumProvider>
            </body>
          </html>
        );
      }
      ```
  </Tab>
  <Tab value="Pages Router">
      ```tsx title="pages/_app.tsx"
      import type { AppProps } from 'next/app';
      // [!code ++:1]
      import { NextJourniumProvider } from '@journium/nextjs';

      export default function App({ Component, pageProps }: AppProps) {
        return (
          {/* [!code ++:1] */}
          <NextJourniumProvider>
            <Component {...pageProps} />
          {/* [!code ++:1] */}
          </NextJourniumProvider>
        );
      }
  ```
  </Tab>
</Tabs>







### Done

You can now start your application and see the Journium SDK in action. If you navigate around in your application, 
the Journium SDK will push telemetry to the Journium API. You can see events in the Journium dashboard in the
**Developers | Events** section of your application. Head over to the 
[Journium dashboard](https://dashboard.journium.app) to see the events.
