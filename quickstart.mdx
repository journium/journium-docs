## Get Journium integrated in 3 simple steps

Start tracking user behavior and getting conversion recommendations immediately.

### Step 1: Create your Journium account

<AccordionGroup>
  <Accordion icon="user-plus" title="Sign up for Journium">
    1. Visit [app.journium.com/signup](https://app.journium.com/signup)
    2. Create your account and verify your email
    3. Create your first project and get your **API key**
    
    <Warning>Keep your API key secure - you'll need it for the next step!</Warning>
  </Accordion>
  <Accordion icon="key" title="Get your project credentials">
    From your Journium dashboard:
    
    1. Navigate to **Project Settings**
    2. Copy your **Project ID** and **API Key**
    3. Note your project's **environment** (development/production)
    
    <Tip>You can create separate projects for development and production environments.</Tip>
  </Accordion>
</AccordionGroup>

### Step 2: Install and initialize Journium

Choose your platform and follow the integration guide:

<AccordionGroup>
  <Accordion icon="js" title="JavaScript/HTML">
    Add Journium to any website with our CDN:
    
    ```html
    <script src="https://cdn.journium.com/sdk/v1/journium.min.js"></script>
    <script>
      Journium.init({
        projectId: 'your-project-id',
        apiKey: 'your-api-key',
        environment: 'development' // or 'production'
      });
    </script>
    ```
    
    Or install via npm:
    
    ```bash
    npm install @journium/sdk
    ```
    
    ```javascript
    import Journium from '@journium/sdk';
    
    Journium.init({
      projectId: 'your-project-id',
      apiKey: 'your-api-key',
      environment: 'development'
    });
    ```
  </Accordion>
  
  <Accordion icon="react" title="React">
    ```bash
    npm install @journium/react
    ```
    
    ```jsx
    import { JourniumProvider } from '@journium/react';
    
    function App() {
      return (
        <JourniumProvider
          projectId="your-project-id"
          apiKey="your-api-key"
          environment="development"
        >
          {/* Your app components */}
        </JourniumProvider>
      );
    }
    ```
  </Accordion>
  
  <Accordion icon="nextjs" title="Next.js">
    ```bash
    npm install @journium/nextjs
    ```
    
    In `pages/_app.js` or `app/layout.js`:
    
    ```jsx
    import { JourniumProvider } from '@journium/nextjs';
    
    export default function MyApp({ Component, pageProps }) {
      return (
        <JourniumProvider
          projectId="your-project-id"
          apiKey="your-api-key"
          environment="development"
        >
          <Component {...pageProps} />
        </JourniumProvider>
      );
    }
    ```
  </Accordion>
</AccordionGroup>

### Step 3: Start tracking and get insights

<Accordion icon="chart-line" title="Verify installation and view analytics">
  1. **Test the integration**: Reload your application and check the browser console for Journium initialization success
  2. **Visit your dashboard**: Go to [app.journium.com](https://app.journium.com) to see real-time data
  3. **View recommendations**: Check the **Insights** tab for AI-powered conversion recommendations
  
  <Info>It may take a few minutes for data to appear in your dashboard after first installation.</Info>
</Accordion>
