---
title: Insights
description: Insights are AI-generated, natural-language analyses of your application's event data, produced by Insight Trackers on a schedule or on demand.
icon: Insight
---

An Insight is the output of a [Insight Tracker](/docs/js/concepts/insight_trackers) run — a concise,
AI-generated summary of your application's event data for a given time window.
